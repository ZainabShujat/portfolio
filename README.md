## Pages

- **Contact**: Contact information and social links

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

- Update personal information in all page files (`app/page.tsx`, `app/about/page.tsx`, etc.)
- Replace placeholder project data in `app/projects/page.tsx`
- Update social links in `components/Footer.tsx`
- Customize colors and animations in `tailwind.config.ts`

## Build for Production

```bash
npm run build
```

This will create a static export in the `out` directory, ready to deploy to any static hosting service.
